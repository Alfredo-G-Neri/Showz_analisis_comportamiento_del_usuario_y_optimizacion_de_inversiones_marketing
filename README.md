# Análisis de Comportamiento de Usuarios y Optimización de Inversión en Marketing

## Descripción del proyecto

Este proyecto analiza el comportamiento de los usuarios de una plataforma digital con el objetivo de evaluar la efectividad de las inversiones en marketing y comprender cómo diferentes canales de adquisición impactan el crecimiento del negocio.

El análisis permite evaluar métricas clave como retención de usuarios, conversión y retorno de inversión en marketing.

---

## Objetivo

Analizar el comportamiento de los usuarios y evaluar el desempeño de las inversiones en marketing para identificar oportunidades de optimización en la estrategia de adquisición de clientes.

---

## Problema de negocio

Las empresas digitales invierten en múltiples canales de marketing para atraer usuarios, pero no siempre es claro cuáles generan mayor valor.

Este análisis busca responder:

- ¿Qué canales generan más usuarios?
- ¿Cuál es el costo de adquisición de clientes?
- ¿Qué tan rentable es cada canal de marketing?

---

## Resultados / Aprendizajes

- Se analizaron métricas clave de adquisición y retención de usuarios.
- Se evaluó el rendimiento de diferentes canales de marketing.
- Se identificaron oportunidades para optimizar la inversión en marketing.
